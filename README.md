# any-starter-cli

用于快速初始化不同项目模块的脚手架

可基于此脚手架进行二次开发，打造内部的初始化模板的脚手架

# 使用

## 安装

```shell
pnpm add any-starter-cli -g

```

## 帮助命令

```shell
starter -h

```

### 查看已知模板列表

```shell
starter list

```

### 初始化项目模板

```shell
starter init

```

# 更新
